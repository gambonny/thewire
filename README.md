# Monorepo Template for Scalable React Apps
This monorepo provides a modern, scalable setup for building React applications with modular packages. It includes:

- PandaCSS for Atomic Styling – A design system powered by PandaCSS with predefined tokens, utilities, and recipes.
- Optimized Developer Experience – Uses Wireit for efficient script execution, ESLint & Prettier for code quality, and Vitest for testing.
- Vite + SWC for Performance – A fast build pipeline optimized for React.
- Strict TypeScript Configuration – Ensuring maintainability and type safety.
